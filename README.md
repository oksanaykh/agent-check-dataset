# 🔍 Dataset Validation Agent

Агент для валидации CSV-датасетов перед анализом данных.  
Автоматически выполняет **9 категорий проверок** и возвращает подробный отчёт с найденными проблемами и рекомендациями по их исправлению.

---

## 📦 Установка

```bash
pip install pandas numpy
```

---

## 🚀 Использование

### Базовый запуск
```bash
python dataset_validator.py path/to/your/file.csv
```

### С указанием ожидаемых колонок
```bash
python dataset_validator.py data.csv --columns order_id customer_id product_id price quantity
```

### Сохранить отчёт в файл
```bash
python dataset_validator.py data.csv --output validation_report.txt
```

### Все опции
```bash
python dataset_validator.py --help
```

## 📋 Что проверяет агент

| # | Категория | Что проверяется |
|---|-----------|-----------------|
| 1 | **Структура** | Загрузка файла, пустые/дублирующиеся/технические колонки, размер датасета |
| 2 | **Типы данных** | Правильность типов, дата как строка, число как object, смешанные типы |
| 3 | **Пропуски** | Количество и доля null, критичные поля, псевдо-пропуски ("N/A", "Unknown") |
| 4 | **Дубликаты** | Полные дубликаты строк, дубли по ID, повторяющиеся транзакции |
| 5 | **Числа** | Отрицательные/нулевые значения, выбросы, допустимость скидок, согласованность суммы |
| 6 | **Категории** | Дубли из-за регистра/пробелов, мусорные значения, высокая кардинальность |
| 7 | **Даты** | Парсинг дат, диапазон, даты из будущего, пробелы во временном ряду |
| 8 | **Ключи / ID** | Уникальность ID, пустые ID, непоследовательный формат, конфликты сущностей |
| 9 | **Бизнес-логика** | Согласованность сумм/скидок/цен, однозначность product→category |

## 🔧 Использование в коде (импорт как модуль)

```python
from dataset_validator import DatasetValidationAgent

agent = DatasetValidationAgent(
    filepath="my_data.csv",
    expected_columns=["order_id", "customer_id", "product_id", "price", "quantity"]
)
report = agent.run()
print(report)

# Или запустить отдельные проверки:
agent._load()
agent.check_structure()
agent.check_nulls()
agent.check_duplicates()
print(agent.report.render())
```

## 🛠 Автоматическое определение типов колонок

Агент автоматически распознаёт колонки по ключевым словам в названиях:

- **Дата**: `date`, `time`, `дата`, `period`, `month`, `year`
- **Числа**: `amount`, `price`, `qty`, `total`, `revenue`, `discount`
- **ID**: `id`, `_id`, `code`, `number`, `num`
- **Категории**: `category`, `type`, `status`, `region`, `method`
- **Критичные поля**: любые из вышеперечисленных, плюс `customer`, `product`, `order`
